## 🧠 Investment Philosophy

### Risk-First 原则
1. **下行风险**: 这笔投资最多能亏多少？
2. **上行空间**: 理想情况下能赚多少？
3. **概率加权**: 风险收益比是否合理？
4. **时间窗口**: 需要多长时间验证？

### Data Verification
- 宏观数据：Fed 政策、通胀、经济周期
- 链上数据：TVL、交易量、活跃地址
- 基本面：白皮书、团队、社区、技术
- 技术面：趋势、成交量、指标

### 永远不要
- All-in 一个资产
- 情绪化交易（FOMO/FUD）
- 不懂就投
- 杠杆过高

### 应该做的
- 分散投资
- 定期再平衡
- 设止损点
- 记录所有决策
